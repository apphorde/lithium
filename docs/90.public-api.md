# Public API surface

Quick reference of the exported identifiers by package. Use `@li3/web` when you want the browser + reactive bundle, or import the specific package for lighter footprints.

## @li3/reactive
- `canBeObserved(object)` mark objects that can be proxied
- `reactive(object, effect)` create a reactive proxy that runs `effect` on mutations
- `unwrap(object)` get the original object back from a reactive proxy
- `signal(value, options?)` create a reactive signal (optionally shallow)
- `effect(fn)` create a readonly computed signal that re-runs `fn`
- `hook(initialValue?)` tuple of signal + setter
- `observer(signal, fn)` subscribe to signal changes
- `isRef(value)`, `isSignal(value)`, `isEffect(value)`, `unref(value)` type guards/utilities
- Types: `SignalInit`, `Signal<T>`, `Effect<T>`

## @li3/runtime
- Lifecycle helpers: `createRuntimeContext(options)`, `activateContext(ctx)`, `createDom(ctx)`
- Options: `setOption(key, value)`, `getOption(key)`
- Plugin hooks: `createDispatcher(hooks)`, `Plugins`, `onSetup`, `onCreateDom`, `onVisitNode`, `onVisitAttribute`, `onInitComponent`, `onUpdateProp`, `onDestroyComponent`
- Props: `getPropValue(ctx, prop, definition)`, `triggerPropUpdate(ctx, prop, oldValue, newValue)`, `createInputRef(ctx, name, signal)`
- State: `ComponentInstance`, `createState(ctx)`
- Context stack: `push(ctx)`, `pop()`, `getCurrentContext()`
- Types: `RuntimeContext`, `CreateRuntimeOptions`, `LifeCycleFunction`, `UpdateLifeCycleFunction`, `LifeCycleObject`, `ComponentDefinition`, `EventEmitFunction`, `AnyFunction`, `Attribute`, `PropDefinition<T>`

## @li3/dom
- Element utilities: `setProperty`, `setEventHandler`, `setClassName`, `setStyle`, `setText`, `setAttribute`, `clearElement`, `defineEventOnElement`, `emitEvent`
- DOM traversal/utilities: `traverseDom`, `getAttributes`, `getAttribute`, `isElement`, `isFragment`, `tpl(html)`, `domReady(fn?)`, `mapContentToSlots`
- Types: `AnyFunction`, `Attribute`

## @li3/browser
- Lifecycle hooks: `onInit`, `onUpdate`, `onDestroy`
- DOM querying: `defineQuery(selector)`
- Events: `defineEvents(eventNames)`, `defineEvent(name)`
- Props: `defineProps(defs)`, `defineProp(name, def)`
- Reactivity aliases: `computed(fn, effect?)`, `ref(value?, options?)`, `shallowRef(value?)`, `watch(signal, fn)`
- DI: `provide(token, provider)`, `inject(token)`
- Component mounting: `createComponent(name, def)`, `mount(element, def, options?)`, `createComponentFromTemplate(template, name?)`, `createApp(template, spec)`, `loadAndParse(url)` (exported as `load`)
- Exported types: see `@li3/runtime`/`@li3/reactive` (re-exported via `@li3/web`)

## @li3/html-parser
- Parsing/packing: `Parser`, `parse(html)`, `pack(doc)`, `unpack(packed)`, `normalize(node)`
- Serialization/compilation: `serialize(node)`, `compile(html)`
- Utilities: `isSelfClosingTag(tag)`
- Types: `ParserAttribute`, `ParserNode`, `ChildNode`, `ElementNode`, `DocumentNode`, `TextNode`, `CommentNode`, `PackedAttributes`, `PackedChildNode`, `PackedElementNode`, `PackedDocumentNode`, `PackedTextNode`, `PackedCommentNode`

## @li3/plugins
- Bindings: `createAttributeBinding`, `createPropertyBinding`, `createClassBinding`, `createStyleBinding`, `createTextNodeBinding`, `createEventBinding`, `attachElementRef`, `templateRef`, `connectDataModel`
- Resources: `addScriptToPage`, `adoptStyleSheet`, `injectStylesheetOnElement`, `loadCss`, `loadScript`
- Host helpers: `applyHostAttributes`, `hostClasses`
- Templates: `templateForOf`, `templateIf`
- Types: `RuntimeContextExtensions`

## @li3/scope
- Expression compilation: `compileExpression(ctx, expression, args?)`, `compileExpressionBlob(...)`, `compileExpressionEval(...)`
- Reactivity helpers: `computedEffect(ctx, expression, effectFn)`
- Modules: `createBlobModule(code, type?)`

## @li3/store
- `createStore(initialState, actions)` returns store helpers: `store` (action dispatchers), `select`, `get`, `transaction`, `events`, `useStore`

## @li3/use
- `useAsyncFetcher(effect?)` async data helper returning `{ data, error, loading, empty, fetch }`

## @li3/sfc
- SFC parsing/build: `parseSFC(source)`, `getComponentCode(name, parsed, withExport?)`, `loadComponent(name, url)`

## @li3/web
- Convenience bundle exporting `hostClasses`, `loadCss`, `loadScript`, `templateRef` plus everything from `@li3/browser` and `@li3/reactive`; also `load` alias for `Component.loadAndParse`
